Display-> It specifies the display behavior(the type of rendering box) of an element

Block elements

Block element-> These elements take full line of code
We can apply custom width and height to these elements
We can also add margin and padding to these elements

When we change the margin , border and padding then the width of the box increases
But when we use border-sizing then the width and height of the box remain constant content will get shrink

Inline elements

These elements try to come in the same line .By default ,its width is equal to the width of the content We cannot change the width , height, to the inline elements We cannot add vertical padding to these elements

This block will works as block so we can give height , width,margin , padding The inline-block value of display used to make a element both inline and block
The element will behave like a inline element but we can add custom properties to it This content will come in the same line because the prev span is inline but we have changed its width and height
