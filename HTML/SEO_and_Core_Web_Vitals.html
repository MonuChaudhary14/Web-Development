SEO -> SEARCH ENGINE OPTIMISATION


Search Engine Optimization (SEO) is the process of improving a website or web page so that it ranks higher in search engine results (like Google, Bing, etc.) when people search for relevant keywords or phrases
Search more about it

Core Web Vitals

Core Web Vitals are a set of specific factors that Google considers important in a webpage’s overall user experience. They’re part of Google’s Page Experience ranking signals, which means they directly affect how well a site performs in search results.

Cumulative Layout Shift (CLS)


What it measures: Visual stability
What it means: Do elements on the page move around unexpectedly as it loads (like buttons shifting or images jumping)?
Good score: Less than 0.1

Largest Contentful Paint (LCP)


What it measures: Loading performance
What it means: How long it takes for the largest visible element (like a banner image or heading text) to load on the screen.
Good score: Under 2.5 seconds

First Input Delay (FID)

This is being replaced by Interaction to Next Paint (INP) as of March 2024.
What it measured: Interactivity (how quickly the site responds after a user clicks or taps)
INP now measures: The overall responsiveness – how quickly the site reacts during real interactions.
Good INP: Under 200 milliseconds
