Form is the structure to collect the user input

        Tags available in form

        form -> It is used to create form for web input

        input -> It is used to take input from multiple ways

        label -> The text attached to the input is label. Due to this when we click on the content of the label we reaches to the id 
        Every label contain a 'for' attribute whose value must be equal to the id of the input

        select -> drop down

        option 

        button

        fieldset -> This is used to group multiple input elements so that we can give captions

        legend -> It is used to give caption to a field set

Place Holder is used to show a content into the input area while the value attribute puts a value in the text area
